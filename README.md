# Airbnb-Market-Analysis
**Short Description:**  "Analyzed Airbnb listing data using SQL and tabalu to uncover pricing trends, host performance, availability patterns, and location-based insights through interactive dashboards."

## Project Overview
This project analyzes Airbnb listing data using SQL and Tableau to uncover valuable insights about pricing, availability, room types, host performance, and neighborhood trends. SQL was used for data cleaning, transformation, and exploratory analysis, while Tableau was used to create interactive dashboards for data visualization and business insights.

## Objectives
- Analyze Airbnb listing performance.
- Identify pricing trends across neighborhoods.
- Examine room type distribution.
- Discover top-performing hosts.
- Evaluate listing availability and occupancy patterns.
- Build an interactive Tableau dashboard for data-driven decision-making.

## Tools & Technologies
- SQL
- Tableau

- Data Cleaning & Transformation
- Data Visualization

The dataset contains Airbnb listings with details such as:

## SQL Analysis
Key analyses performed using SQL:
- Average price by neighborhood
- Room type distribution
- Top hosts by number of listings
- Availability analysis
- Review trend analysis
- Price comparison across locations

## Tableau Dashboard Features
- KPI Overview
- Price Distribution Analysis

- Room Type Breakdown
- Host Performance Analysis

## Key Insights
- Identified neighborhoods with the highest average prices.
- Compared room type popularity across locations.
- Analyzed host performance and listing concentration.
- Evaluated availability trends and booking potential.
- Visualized market trends through interactive dashboards.
